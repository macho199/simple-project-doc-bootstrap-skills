- 불필요한 파일을 만들지 않는다.
- 기존 패턴과 구조를 먼저 파악한 뒤 수정한다.

## 문서 관리 규칙
- 기본 문서는 {RULE_DOC_NAME}, PROJECT.md, STATUS.md, ARCHITECTURE.md만 유지한다.
- 새 문서를 만들기 전에 기존 문서에 통합 가능한지 먼저 검토한다.
- 오래된 내용은 누적하지 말고 현재 기준으로 갱신한다.
- {RULE_DOC_NAME}에는 모든 세션에 공통 적용되는 규칙만 유지한다.
